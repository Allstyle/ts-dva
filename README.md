ts-dva
=======

TypeScript + dva + Antd

Install[安装依赖]
-------

```bash
node >= 5.0.0
```
```bash
$ npm install -g typings
$ npm install -g typescript
$ typings install
$ npm install
```

Usage[用法]
-------

```bash
$ npm start
```
